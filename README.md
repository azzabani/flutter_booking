# Système de Gestion de Stock

## Description
Application web de gestion de stock développée avec ASP.NET Core MVC.

## Fonctionnalités
- Gestion des produits
- Gestion des fournisseurs
- Gestion des catégories
- Suivi des mouvements de stock
- Gestion des commandes
- Système d'alertes

## Technologies utilisées
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap

## Installation
1. Cloner le repository
2. Configurer la chaîne de connexion dans appsettings.json
3. Exécuter les migrations : `dotnet ef database update`
4. Lancer l'application : `dotnet run`
