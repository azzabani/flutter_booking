# Changelog

Toutes les modifications notables de ce projet seront documentées dans ce fichier.

## [1.0.0] - 2026-04-12

### Ajouté
- Interface utilisateur moderne avec Material Design 3
- Système d'authentification Firebase
- Gestion complète des réservations
- Dashboard administrateur avec statistiques
- Calendrier interactif pour la sélection de créneaux
- Système de notifications in-app
- Génération de PDF pour les confirmations
- Navigation avec Bottom Navigation Bar

### Fonctionnalités
- **Authentification** : Connexion/inscription sécurisée
- **Réservations** : Création, modification, annulation
- **Administration** : Validation des réservations, gestion des ressources
- **Profil** : Gestion des informations utilisateur
- **Notifications** : Alertes en temps réel

### Technique
- Flutter 3.x avec SDK Dart ^3.9.2
- Firebase (Auth, Firestore, Core)
- Provider pour la gestion d'état
- Table Calendar pour l'interface calendrier
- PDF et Printing pour la génération de documents
